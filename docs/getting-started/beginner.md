## 🌉 다리 1 — 코드 시운전 (1분)

자동차 시동만 한 번 걸어보는 거예요. 어디 가지는 않아요.

PowerShell을 켜고 (윈도우 키 + R → `powershell` → 엔터):

```powershell
cd C:\Users\jooladen\Desktop\stock\morning_us_index
pip install -r requirements.txt
pip install pytest
pytest
```

마지막 줄에 **`16 passed`** 가 보이면 성공이에요. 🎉

> 빨간 글씨가 보여도 당황 마세요. 대부분 `pip` 업데이트 권유 같은 알림이에요. **`passed`** 라는 단어만 찾으면 돼요.

## 🌉 다리 2 — 슬랙 "우체통" 만들기 (5분)

**Webhook URL** = 슬랙에 메시지를 넣어주는 **비밀 우체통 주소**예요. 이 주소를 알면 누구든 메시지를 넣을 수 있어요.

1. https://api.slack.com/apps 접속
2. 초록색 **Create New App** 버튼 → **From scratch** 선택
3. App Name: `morning-us-index` (마음대로) — 워크스페이스: 메시지 받을 곳 선택 → **Create App**
4. 왼쪽 메뉴에서 **Incoming Webhooks** 클릭
5. 오른쪽 위 **Off** 토글을 **On** 으로 누르기
6. 페이지 **맨 아래로** 스크롤 → **Add New Webhook to Workspace** 버튼
7. 메시지 받을 채널 선택 (예: `#트레이딩` 또는 본인 DM) → **Allow**
8. 화면에 **`https://hooks.slack.com/services/T..../B..../xxxxxxxxxxxx`** 같은 긴 주소가 나와요
9. **이 주소를 복사해서 메모장에 잠깐 보관** ← ⚠️ 다음 단계에서 씀

> 🚨 **이 주소는 비밀번호와 같아요.** 누구한테도 보여주지 마세요. 카카오톡/이메일/스크린샷에 절대 올리지 마세요. 새어나가면 다시 8번에서 새로 발급받으면 돼요.

## 🌉 다리 3 — 깃허브에 올리기 + 비밀번호 등록 (10분)

### 3-1. 깃허브에 빈 저장소 만들기

1. https://github.com/new 접속
2. **Repository name**: `morning-us-index`
3. **Public** 선택 (Private도 되지만 Public이 무료 한도 더 넉넉)
4. 아래 **Add a README**, **.gitignore**, **license** 모두 **체크 해제** (이미 코드에 있어서 충돌 방지)
5. **Create repository** 버튼

### 3-2. 코드 올리기 (PowerShell)

```powershell
cd C:\Users\jooladen\Desktop\stock\morning_us_index
git init
git add .
git commit -m "init: morning-us-index Phase 1"
git branch -M main
git remote add origin https://github.com/<당신의-아이디>/morning-us-index.git
git push -u origin main
```

`<당신의-아이디>` 부분을 본인 깃허브 ID로 바꿔주세요. (예: `https://github.com/jooladen/morning-us-index.git`)

처음 push할 때 깃허브 로그인 창이 뜨면 로그인하면 돼요.

성공하면 깃허브 페이지에 파일들이 보여요.

### 3-3. 비밀번호(Webhook URL) 등록

1. 방금 만든 깃허브 저장소 페이지에서 **Settings** 탭 (저장소 안쪽 Settings, 개인 Settings 아님)
2. 왼쪽 메뉴 **Secrets and variables** → **Actions** 클릭
3. 초록색 **New repository secret** 버튼
4. **Name**: `SLACK_WEBHOOK_URL` (대문자, 밑줄 정확히 — 이거 틀리면 안 돼요)
5. **Secret**: 다리 2의 9번에서 메모장에 보관한 긴 주소 **전체** 붙여넣기
6. **Add secret**

> 메모장에 적은 URL은 이제 지워도 돼요. 깃허브 안에 안전하게 저장됐어요.

## 🤔 자주 묻는 질문

**Q. 평일에 한국 공휴일에도 메시지가 오나요?**
A. 네. 미국 시장이 열려 있으면 보내요.

**Q. 미국이 휴장(토/일/공휴일)이면 어떻게 되나요?**
A. 마지막 거래일의 종가를 보내고, "(미 증시 휴장 / 마지막 거래일)" 표시가 같이 와요.

**Q. 6시 정각이 아니라 6시 17분에 와요. 정상?**
A. 정상이에요. GitHub Actions의 시간은 5–15분 정도 늦을 수 있어요. 06:00–06:30 윈도우면 OK.

**Q. 한 달에 돈이 나가나요?**
A. 안 나가요. GitHub Public 저장소 + Slack 무료 플랜이면 한도 안에서 무제한.

**Q. 다른 지수(다우, KOSPI 등)도 추가하고 싶어요.**
A. `config.py`의 `TICKERS` 리스트에 한 줄 추가하면 돼요. 예: `("^DJI", "다우")`.

**Q. AI 분석도 같이 받고 싶어요.**
A. Phase 2에서 만들 거예요. 이번엔 숫자만, 다음 단계에서 AI가 "왜 올랐는지" 설명해주는 기능 추가 예정.
