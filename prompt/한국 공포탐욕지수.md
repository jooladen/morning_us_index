● 한국 공포탐욕 지수 계산법

  🟢 ─초딩용: 시장이 무섭냐 신나냐를 숫자로 표현하는 것. 여러 지표를 점수화해서 평균 냄.

  🔵 개발자용: CNN F&G와 동일한 구조, 한국 시장 지표로 대체.

  ---
  주로 쓰는 7가지 구성 요소 (가중 평균):

  ┌─────────────┬────────────────────────┬────────────────────┐
  │    지표     │       한국 버전        │   미국 CNN 버전    │
  ├─────────────┼────────────────────────┼────────────────────┤
  │ 변동성      │ VKOSPI                 │ VIX                │
  ├─────────────┼────────────────────────┼────────────────────┤
  │ 모멘텀      │ KOSPI vs 125일 MA      │ S&P vs 125일 MA    │
  ├─────────────┼────────────────────────┼────────────────────┤
  │ 풋/콜 비율  │ KOSPI 옵션 P/C ratio   │ CBOE P/C ratio     │
  ├─────────────┼────────────────────────┼────────────────────┤
  │ 시장 강도   │ 상승종목/하락종목 비율 │ NYSE 신고가/신저가 │
  ├─────────────┼────────────────────────┼────────────────────┤
  │ 외국인 수급 │ 외국인 순매수 3주 추세 │ (없음)             │
  ├─────────────┼────────────────────────┼────────────────────┤
  │ 공매도      │ 공매도 비중            │ Junk bond demand   │
  ├─────────────┼────────────────────────┼────────────────────┤
  │ 거래량      │ 거래량 vs 20일 평균    │ Safe haven demand  │
  └─────────────┴────────────────────────┴────────────────────┘

  각 지표를 0~100으로 정규화 후 가중 평균 → 최종 점수.

  ---
  게이지 + 라인 차트 렌더링

  🟢  초딩용: 바퀴처럼 생긴 반원 그래프는 특별한 그리기 도구로 만든 것.

  🔵 개발자용:

  - 게이지: SVG arc + 바늘 회전 (D3.js 또는 순수 SVG). 각도 = (value / 100) * 180 - 90도
  - 라인 차트: Chart.js 또는 Recharts — 1년치 일별 데이터 히스토리를 DB에 쌓아두고 렌더링
  - 색상 그라데이션: 0(빨강) → 50(노랑) → 100(초록) CSS linear-gradient로 arc에 입힘
