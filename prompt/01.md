나스닥(^IXIC)과 S&P500(^GSPC) 지수를 분석하여 매일 아침 6시(KST)에 슬랙으로 리포트를 보내는 자동화 시스템을 구축해줘.

### 요구사항:
1. 데이터 수집: 
   - yfinance 라이브러리를 사용하여 두 지수의 종가와 전일 대비 변동률을 가져올 것.
   - [MCP Tool 활용] Google Search나 Fetch MCP를 사용해 '어제 미국 증시 상승/하락 원인'에 대한 최신 뉴스 헤드라인 3~5개를 수집해줘.

2. AI 분석 (Reasoning):
   - Gemini API(또는 Claude API)를 사용하는 파이썬 스크립트(`main.py`)를 작성해줘.
   - 수집된 지수 데이터와 뉴스 내용을 AI에게 전달하여 "왜 시장이 이렇게 움직였는지" 전문적인 브리핑 문구(슬랙용 마크다운 형식)를 생성하게 할 것.

3. 자동화 배포:
   - 내 컴퓨터를 꺼도 작동하도록 GitHub Actions 워크플로우 파일(`.github/workflows/daily_report.yml`)을 작성해줘.
   - 한국 시간 오전 6시(UTC 21:00)에 실행되도록 cron을 설정해줘.
   - API Key와 Slack Webhook URL은 GitHub Secrets(`GEMINI_API_KEY`, `SLACK_WEBHOOK_URL`)에서 불러오도록 코드를 짜줘.

4. 최종 출력물:
   - `main.py` (파이썬 로직)
   - `requirements.txt` (필요 라이브러리)
   - `daily_report.yml` (깃허브 액션 설정)
   - 설치 및 실행 가이드 (README.md)

자, 이제 MCP를 활용해서 뉴스부터 검색해보고 코드 작성을 시작해줘!
